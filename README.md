# 🩺 Health Survey Form

This is a simple and responsive Health Survey Form built using HTML and CSS.

## 🚀 Features
- User-friendly form design
- Input fields for personal details
- Radio buttons for gender and medical conditions
- Checkboxes for selecting symptoms
- Responsive layout using Flexbox

## 🛠️ Technologies Used
- HTML5
- CSS3

## 📌 Purpose
This project is created to practice HTML forms, input types, and layout design.
